
# Priority Scheduler

This is our OS Project which shows how Priority Scheduling works in CPU.




## Tech Stack

**Client:** React, Redux, TailwindCSS, TypeScript

**Server:** Node, Express


## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/AshishKumar25x/PriorityScheduler.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
